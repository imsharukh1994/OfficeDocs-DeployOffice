# Microsoft 365 Companions apps overview

Microsoft 365 Companions apps provide users with tools to streamline their workflows by enhancing accessibility to key features such as **People search** and **File search**. These apps integrate directly into Microsoft 365 to improve productivity and efficiency.

## Availability

Microsoft 365 Companions apps will be available to users through the Microsoft 365 ecosystem. Details on supported platforms and rollout timelines will be provided closer to release.

## How to opt out

Administrators can opt out of Microsoft 365 Companions apps deployment through the Microsoft 365 admin center. Instructions for disabling these apps will be provided to ensure organizations have control over their deployment.

## What happens if admins don’t opt out within 30 days?

If administrators do not opt out within 30 days of availability, the Microsoft 365 Companions apps will remain active in the tenant. Organizations that wish to remove these apps afterward will need to follow a set of clean-up steps, which will be outlined in Microsoft documentation.

## Providing feedback

### Admin feedback

Administrators can submit feedback through the Microsoft 365 admin center using built-in feedback tools.

### User feedback

End users can provide feedback directly within the Microsoft 365 Companions apps via in-app feedback options.

## Next steps

For additional details and updates, refer to Microsoft 365 documentation as more information becomes available.
